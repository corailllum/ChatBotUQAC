# ChatBotUQAC 
## But du projet :
Le but est de mettre en place un chatbot fonctionnel pour répondre aux questions portant sur la documentation de l'UQAC
## Technologie utilisées :
- **Librairies :** LangGraph, LangChain, Python LangGraph, tempfile, Streamlit 
- **Techniques :** RAG, webscrapping (liens pertinents + mise en mémoire) 
- **Modèles :** GPT4ALL (https://www.nomic.ai/gpt4all), Llama, prendre un petit modèle 
## Organisation du Git :
